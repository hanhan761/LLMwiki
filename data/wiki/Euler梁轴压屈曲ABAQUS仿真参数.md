# Euler梁轴压屈曲问题 — ABAQUS仿真参数手册

## 问题描述
薄弹性Euler-Bernoulli梁在轴向均匀压缩下的屈曲及后屈曲响应，涉及三种边界条件对应三个算例。

## 材料参数
- **杨氏模量**：$E = 1000$ nN/nm²（即1 TPa）
- **泊松比**：$\nu = 0$（一维梁问题，泊松比不影响结果）

## 几何参数
- **梁长**：$L = 50$ nm
- **截面宽度**：$b = 1.0$ nm
- **截面厚度**：$h = 0.5$ nm
- **截面积**：$A = bh = 0.5$ nm²
- **惯性矩**：$I = bh^3/12 = 0.01042$ nm⁴
- **截面类型**：矩形实心截面

## 加载方式
- **加载类型**：轴向压缩位移（位移控制）
- **应变扫描范围**：
  - 简支梁：应变$0 \sim 0.01$，对应位移$0 \sim 0.5$ nm
  - 固支梁：应变$0 \sim 0.05$，对应位移$0 \sim 2.5$ nm
  - 周期边界：应变$0 \sim 0.05$，对应位移$0 \sim 2.5$ nm
- **关键设置**：必须开启几何非线性（**Nlgeom = ON**）

## 边界条件
### 简支梁（KAN_SIMPLE）
- 两端Pin支座：约束平动，释放转动
- 一端轴向固定，另一端施加压缩位移

### 固支梁（KAN_CLAMPED）
- 两端Encastre：全固定
- 右端释放轴向自由度改为位移加载

### 环形梁/周期边界（KAN_CIRCLE）
- 物理上为封闭圆环，均匀环向压缩
- 建模方式：直接建圆环或直梁加周期耦合

## 理论临界值
### 临界应变
- 简支梁：$\varepsilon_c = \dfrac{\pi^2 I}{A L^2} = 8.22 \times 10^{-5}$
- 固支梁：$\varepsilon_c = \dfrac{4\pi^2 I}{A L^2} = 3.29 \times 10^{-4}$
- 周期边界：$\varepsilon_c = \dfrac{4\pi^2 I}{A L^2} = 3.29 \times 10^{-4}$

### 临界力
- 简支梁：$P_c = 0.0411$ nN
- 固支梁：$P_c = 0.1645$ nN
- 周期边界：$P_c = 0.1645$ nN

### 一阶屈曲模态形状
- 简支梁：半波正弦
- 固支梁：全波余弦
- 周期边界：全波正弦

## ABAQUS分析步建议
### Step 1：线性屈曲分析（Buckle）
- 类型：Linear perturbation → Buckle
- 目的：提取屈曲特征值和模态，验证模型
- 提取前3~5阶模态

### Step 2：后屈曲分析（Static, Riks）
- 类型：Static, Riks（弧长法）
- 开启几何非线性（Nlgeom = ON）
- 初始缺陷：用一阶模态作为几何缺陷，幅值取梁厚的1%~10%
- 目的：追踪后屈曲载荷-位移曲线

## 后屈曲标度律
在临界点附近，最大横向挠度满足$w_\text{max} \propto \sqrt{\varepsilon - \varepsilon_c}$，可用于验证仿真结果。

## 单位制说明
采用{nm, nN, nN/nm²}单位制，确保ABAQUS中数值自洽。
